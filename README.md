# AI-Assisted React Study Planner

A simple React study planner app that demonstrates a beginner-friendly frontend project. Users can add tasks, assign subjects, choose due dates, mark tasks as completed, filter tasks, and keep data saved with localStorage.

## Features

- Add a study task
- Choose a subject
- Set a due date
- Mark tasks as complete or pending
- Delete tasks
- Filter by All / Pending / Completed
- Task summary counts
- Saves tasks in localStorage

## Tech Stack

- React
- Vite
- CSS
- localStorage

## How to Run

1. Open the project in VS Code
2. Open the terminal in the project folder
3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open the local URL shown in the terminal
